# krotka_runner

GitHub Actions automation for managing and maintaining a self-hosted GitHub Actions runner on Ubuntu 24.04.

## Infrastructure

| Host | IP | Role |
|------|----|------|
| runner | 192.168.1.204 | Self-hosted GitHub Actions runner |
| lsrv-01 | 192.168.1.201 | General purpose server (future use) |

User: `piotr`

## Workflows

| Workflow | Schedule | Purpose |
|----------|----------|---------|
| `github_runner_selfhosted.yml` | 2:00 AM UTC daily | apt updates, hardening, health checks, log cleanup |
| `runner-nightly-reboot.yml` | 3:00 AM UTC daily | Reboot runner after maintenance |

## GitHub Secrets Required

| Secret | Description |
|--------|-------------|
| `RUNNER_SSH_PRIVATE_KEY` | ed25519 private key for SSH access to 192.168.1.204 |
| `LSRV01_PASSWD` | Password for piotr on lsrv-01 (192.168.1.201) |

## GitHub Variables (optional)

| Variable | Default | Description |
|----------|---------|-------------|
| `DISK_THRESHOLD` | `85` | Disk usage % that fails the health check |
| `MEM_THRESHOLD` | `90` | Memory usage % that triggers a warning |

## SSH Key Setup (one-time)

### Initial setup

```bash
# 1. Generate keypair (no passphrase)
ssh-keygen -t ed25519 -C "github-actions-runner" -f ~/.ssh/runner_key

# 2. Copy public key to runner host (uses password auth one last time)
ssh-copy-id -i ~/.ssh/runner_key.pub piotr@192.168.1.204

# 3. Test key login
ssh -i ~/.ssh/runner_key piotr@192.168.1.204
# Should log in without password prompt

# 4. Configure passwordless sudo on the runner host (while logged in)
sudo bash -c "echo 'piotr ALL=(ALL) NOPASSWD: ALL' > /etc/sudoers.d/piotr-nopasswd"
sudo chmod 440 /etc/sudoers.d/piotr-nopasswd

# 5. Add private key to GitHub Secret RUNNER_SSH_PRIVATE_KEY
cat ~/.ssh/runner_key
# Copy full output → GitHub repo → Settings → Secrets and variables → Actions → New secret
```

### Save private key to LastPass

1. Run `cat ~/.ssh/runner_key` and copy the full output
2. Open LastPass → **+** → **Secure Note**
3. Name: `runner_key - github-actions-runner`
4. Paste key into **Notes** field → Save

### Restore key on a new machine

```bash
# 1. Copy private key content from LastPass secure note, then:
cat > ~/.ssh/runner_key << 'EOF'
-----BEGIN OPENSSH PRIVATE KEY-----
<paste your key from LastPass here>
-----END OPENSSH PRIVATE KEY-----
EOF

# 2. Set correct permissions
chmod 600 ~/.ssh/runner_key

# 3. Regenerate public key from private key
ssh-keygen -y -f ~/.ssh/runner_key > ~/.ssh/runner_key.pub

# 4. Test
ssh -i ~/.ssh/runner_key piotr@192.168.1.204
```
